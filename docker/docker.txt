1. download and install docker: https://hub.docker.com/editions/community/docker-ce-desktop-windows
	- if not windows10 pro -> https://docs.docker.com/toolbox/toolbox_install_windows/
	- test: docker --version

2. download tomcat image:
	- docker pull tomcat

3. config and run tomcat directory with war file
	- create directory "java" with Dockerfile
	- cd to directory "java"
	- docker build --tag mytomcatserver .
	- docker run -it --rm -p 8888:8080 mytomcatserver

4. test:
	- in browser: http://192.168.99.100:8888
	- curl: curl http://192.168.99.100:8888/TareasProyectosJersey/rest/usuarios

5. publish:
	- docker login
	- docker tag mytomcatserver <docker hub <username>/mytomcatserver:part1
	- docker push <username>/mytomcatserver:part1
